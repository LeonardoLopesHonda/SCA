# Cloning Project
 
## Windows

```ps1
    git clone https://github.com/rvfrancozo/SCA.git
```

```ps1
    cd sca
```

```ps1
    cp .env.example .env
```

```ps1
    nano .env
```

```ps1
    composer install
```

```ps1
    php artisan key:generate
```

```ps1
    php artisan migrate --seed
```
